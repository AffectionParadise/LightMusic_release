# 基于Swing与JavaFx的音乐播放器——轻音

## 软件特性：
- 支持本地音乐管理
- 在线音乐收听（音频来源包含：网易云、酷狗、腾讯、酷我、咪咕、千千）
- 支持歌单、专辑、歌手、电台搜索
- MV 与视频收看（视频来源包含：网易云、酷狗、腾讯、酷我、千千、猫耳、B站）
- 音乐与视频下载功能
- 更多小功能等你探索~
